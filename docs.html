Introduction

Selamat datang di dokumentasi Loruki! Platform kami dirancang untuk membantu pengelolaan aplikasi secara efisien, baik untuk pemula maupun profesional. Dalam dokumentasi ini, Anda akan menemukan panduan lengkap untuk memulai perjalanan Anda dengan Loruki

Loruki menyediakan solusi cepat dan mudah untuk mengelola aplikasi Anda dengan performa maksimal. Temukan fitur terbaik kami sekarang juga!

Loruki CLI adalah alat baris perintah yang memungkinkan Anda mengelola aplikasi dengan lebih cepat. Instalasi mudah dan panduan penggunaannya disediakan dalam dokumentasi ini

Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install loruki-cli

NPM

$ npm install loruki-cli

Yarn

$ yarn install loruki-cli

Pastikan perangkat Anda memiliki koneksi internet stabil untuk proses instalasi dan pengelolaan menggunakan Loruki CLI.
